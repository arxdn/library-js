# @arxdn/library-js

Monorepo for JavaScript libraries from ARXDN.

## Packages

- **@arxdn/shared** - Shared utilities and common types
- **@arxdn/toast** - Toast notification system

## Scripts

```bash
# Build all packages
npm run build

# Run tests
npm run test

# Linting
npm run lint

# Format code
npm run format

# Type checking
npm run typecheck
```

## Structure

```
├── packages/
│   ├── shared/         # Shared utilities
│   └── toast/          # Toast notification library
├── sites/
│   └── docs/           # Documentation site
└── package.json        # Monorepo configuration
```
